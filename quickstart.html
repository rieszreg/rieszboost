Quickstart — ATE in 30 lines

This page fits a Riesz representer for the average treatment effect on a synthetic dataset. Pick the language tab on any code block; your choice persists across the site.

Simulate

Binary treatment with known propensity \(\pi(x)\). We will recover \(\alpha_0(a, x) = a/\pi(x) - (1-a)/(1-\pi(x))\) without estimating \(\pi\).

  • Python
  • R
import numpy as np
import pandas as pd
from rieszboost import RieszBooster, ATE

rng = np.random.default_rng(0)
n = 4000
x = rng.uniform(0, 1, n)
pi = 1 / (1 + np.exp(-(-0.02 * x - x**2 + 4 * np.log(x + 0.3) + 1.5)))
a = rng.binomial(1, pi)
df = pd.DataFrame({"a": a.astype(float), "x": x})

Fit

Configure the booster on the constructor, then call .fit(df). Early stopping uses an internal validation split sized by validation_fraction.

  • Python
  • R
booster = RieszBooster(
    estimand=ATE(treatment="a", covariates=("x",)),
    n_estimators=2000,
    learning_rate=0.05,
    max_depth=4,
    early_stopping_rounds=20,
    validation_fraction=0.2,
)
booster.fit(df)

Predict and compare to the closed form

We know \(\alpha_0\) here, so we can score the fit directly.

  • Python
  • R
alpha_hat  = booster.predict(df)
true_alpha = a / pi - (1 - a) / (1 - pi)
print(f"corr(α̂, α₀) = {np.corrcoef(alpha_hat, true_alpha)[0, 1]:.3f}")
corr(α̂, α₀) = 0.934
print(f"RMSE         = {np.sqrt(np.mean((alpha_hat - true_alpha)**2)):.3f}")
RMSE         = 0.973
alpha_hat  <- booster$predict(df)
true_alpha <- a / pi - (1 - a) / (1 - pi)
cat(sprintf("corr(alpha_hat, alpha_0) = %.3f\n",
            cor(alpha_hat, true_alpha)))
corr(alpha_hat, alpha_0) = 0.962
cat(sprintf("RMSE                     = %.3f\n",
            sqrt(mean((alpha_hat - true_alpha)^2))))
RMSE                     = 0.768

Visual sanity check

  • Python
  • R
import matplotlib.pyplot as plt
fig, ax = plt.subplots()
ax.scatter(true_alpha, alpha_hat, s=4, alpha=0.4)
lim = max(abs(true_alpha).max(), abs(alpha_hat).max())
ax.plot([-lim, lim], [-lim, lim], "k--", lw=1)
ax.set_xlabel(r"$\alpha_0$ (closed-form)")
ax.set_ylabel(r"$\hat\alpha$ (rieszboost)")
plt.show()
lim <- max(abs(c(true_alpha, alpha_hat)))
plot(true_alpha, alpha_hat, pch = 16, cex = 0.4, col = adjustcolor("black", 0.4),
     xlim = c(-lim, lim), ylim = c(-lim, lim),
     xlab = expression(alpha[0]~"(closed-form)"),
     ylab = expression(hat(alpha)~"(rieszboost)"))
abline(0, 1, lty = 2)
